# Assets Folder

## Sounds
Place your sound files in the `sounds/` folder:

- `correct.mp3` - Positive beep for correct clear
- `wrong.mp3` - Error buzz for friendly fire
- `combo.mp3` - Rising sound for combos
- `gameStart.mp3` - Game start sound
- `gameOver.mp3` - Game over sound

## Where to Get Sounds

### Freesound.org (Recommended)
1. Go to https://freesound.org
2. Search for:
   - "positive beep" (filter: CC0)
   - "error buzz" (filter: CC0)
   - "success fanfare short" (filter: CC0)
   - "game over sound 8-bit" (filter: CC0)
3. Download MP3 format
4. Place in this folder

### Kenney.nl
- Visit: https://kenney.nl/assets/digital-audio
- Download "Digital Audio" pack
- Extract UI sounds

### Mixkit.co
- Visit: https://mixkit.co/free-sound-effects/
- Search for game/UI sounds
- Download and place here

## Note
The game will work without sound files - they're optional enhancements.
