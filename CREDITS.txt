SOUND EFFECTS
================================================================================

All sound effects are sourced from Freesound.org under Creative Commons licenses.

Source: https://freesound.org
License: Creative Commons (CC0 or CC BY - see individual file attributions below)

Sound Files Used:
------------------
- correct.mp3    - Positive feedback sound for correct enemy utility clears
- wrong.mp3      - Error sound for friendly fire (clicking friendly utilities)
- combo.mp3      - Rising sound effect for combo multipliers
- start.mp3      - Game start sound/fanfare
- stop.mp3       - Game over sound effect

Note: All sounds were filtered by "Creative Commons 0" (CC0) license when 
possible, which requires no attribution. If any sounds use CC BY license, 
attribution is provided below.
